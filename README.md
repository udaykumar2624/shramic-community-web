# Shramic Blog Application

## 🚀 Overview
This is a full-stack blog application developed using Next.js, Node.js, and MongoDB Atlas.  
It allows users to view blog posts and dynamically fetch content from a backend API.

---

## 🛠️ Tech Stack
- Frontend: Next.js, React, Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB Atlas  

---

## 📁 Project Structure
- frontend/ → Next.js frontend  
- backend/ → Express backend  

---

## ⚙️ Setup Instructions

### 1. Run Backend
cd backend  
npm install  
npx nodemon server.js  

### 2. Run Frontend
cd frontend  
npm install  
npm run dev  

---

## 🌐 Access
Frontend: http://localhost:3000  
Backend API: http://localhost:5000/api/blog/posts  

---

## 📌 Features
- Blog listing page  
- Dynamic blog pages (slug-based routing)  
- API integration  
- MongoDB database storage  

---

## 🧠 Note
Ensure MongoDB connection is configured using environment variables.
